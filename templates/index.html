Welcome to Github Stats

The GitHub Stats project is a data visualization tool that retrieves and displays GitHub repository information and commit statistics. Using a Flask backend, it communicates with the GitHub API to fetch repository data and commits, presenting them in an organized and user-friendly manner on the frontend. The project aims to provide insights into GitHub project activities for enhanced analysis and tracking
